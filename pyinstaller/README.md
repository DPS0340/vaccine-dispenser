PyInstaller for CI Builds
=========================

CI 빌드에서만 사용하는 파일입니다.
소스코드 다운로드시 오진 문제로 두 파일을 암호화 하여 압축했습니다. (See SJang1/korea-covid-19-remaining-vaccine-macro#524)
패스워드는 `pyinstaller` 입니다.

빌드는 Github Actions를 통해서 이뤄졌습니다.
